Transcription audio en .srt

Version simple : 2 boutons, upload d'un fichier, transcription Whisper, téléchargement du SRT.

Page ouverte en fichier local (file://) — l'extraction audio (MP4) utilise un module qui ne fonctionne pas dans ce mode (origine « null », Web Workers bloques). Lance un petit serveur local puis ouvre la page en http://localhost (ex. npx --yes serve . ou python -m http.server 8080). La transcription avec envoi du fichier complet (MP4) peut quand meme fonctionner.
Glisser-deposer un fichier (MP4/MP3)

Ou clique ici pour choisir un fichier. Ensuite, lance la transcription.

Envoie seulement la piste son au lieu de toute la vidéo : fichier plus léger, upload souvent plus court. Nécessite npm install dans le dossier du projet (dossier node_modules à côté de index.html). Premier chargement : lecture des fichiers FFmpeg depuis ton serveur local.
